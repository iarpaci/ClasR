This document classifies all CLASR-EN kits
into three tiers by functional importance and maintenance priority.

Purpose:
  Clarify what is untouchable

  Does not modify operational order
  Is a classification and maintenance guide only

## 5. NEW KIT ADDITION PROTOCOL

Before adding a new kit, answer:
1. Which tier does this kit belong to? (typically Tier 3)
2. Which SECTION does it connect to? (update LENS-BRIDGE if needed)
3. Does it add a new row to the canonical source table?
4. Does the VERSION FREEZE need updating?

Do not add a kit without answering all four.

## 2. RATIONALE FOR v1.5 ADDITIONS

Two signal gaps identified through architecture review:

Gap 1 — Novelty claim behavior untracked:
  The system had no dedicated layer for reading
  how manuscripts assert originality and contribution.
  "First study," "novel," "to our knowledge" are structurally
  significant signals that Q1 reviewers actively verify.
  Orientation Lens captured some framing closure behavior,
  but contribution claim intensity and boundedness
  were not independently mapped.
  → Closed by Contribution Framing Kit v1.0.

Gap 2 — Results-to-Discussion drift untracked:
  The system read argumentation chain continuity (SECTION 4)
  and scale transition in numerical outputs (LENSSA-DEPTH),
  but the interpretive expansion from Results to Discussion —
  causal drift, scale drift, mechanism introduction,
  implication extension — was not systematically mapped
  as a structural drift pattern with magnitude.
  → Closed by Discussion Scope Drift Kit v1.0.
