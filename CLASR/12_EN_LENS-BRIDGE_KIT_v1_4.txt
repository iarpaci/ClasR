# EN — ARGUMENT-DENSITY KIT v1.0

---

## 0. IDENTITY

Type: EXTENSION KIT
Attaches to: UNIFIED-OUTPUT KIT v1.0 (before closing block)
No judgment · No recommendation · No rewriting

This kit maps the distribution of claim load
across manuscript sections as a cross-section signal layer.

---

## 1. DESIGN PRINCIPLE

Existing lenses read each signal tied to one SECTION.
This kit operates differently:

```
After all SECTION outputs are produced
         ↓
Signal count per SECTION is aggregated
         ↓
Load distribution is mapped
         ↓
Single block displayed before the closing block
```

Does not touch SECTION order.
Does not open new SECTIONs.
Re-reads lens outputs; does not re-produce them.

---

## 2. MEASURED ZONES

The manuscript is evaluated across four functional zones:

```
FRAMING   → SECTION 0 + SECTION 1 + SECTION 2
CORE      → SECTION 3 + SECTION 4 + SECTION 5
LANGUAGE  → SECTION 6
CLOSING   → SECTION 7 + SECTION 8
```

---

## 3. DENSITY MEASUREMENT

For each zone, active signal counts from relevant SECTIONs are aggregated:
- Verbal Lens signals
- LENSSA-DEPTH patterns
- FREQUENCY SIGNAL outputs
- Temporality / Silence / Orientation / Integrity signal detections
- Citation Behavior signals
- Abstract–Body Coherence signals
- Replication Signal profile gaps
- Contribution Framing signals
- Discussion Scope Drift signals

Signal count → zone load:
```
0–1 signals  → LOW
2–3 signals  → MEDIUM
4+  signals  → HIGH
```

---

## 4. DISTRIBUTION PROFILES

Four primary distribution patterns recognized:

**FRONT-LOADED:**
FRAMING zone HIGH, CLOSING zone LOW
→ Strong promise; quiet closing

**BACK-LOADED:**
FRAMING zone LOW, CLOSING zone HIGH
→ Modest opening; overstated conclusion

**CORE-CONCENTRATED:**
CORE zone HIGH, others LOW or MEDIUM
→ Load concentrated in method and argumentation

**DISTRIBUTED:**
All zones MEDIUM or balanced
→ Load spread evenly across the manuscript

Multiple profiles may be detected simultaneously.

---

## 5. OUTPUT FORMAT

Displayed immediately before the closing block:

```
▸ ARGUMENT DENSITY

FRAMING:  [LOW / MEDIUM / HIGH]
CORE:     [LOW / MEDIUM / HIGH]
LANGUAGE: [LOW / MEDIUM / HIGH]
CLOSING:  [LOW / MEDIUM / HIGH]

Distribution: [FRONT-LOADED / BACK-LOADED /
               CORE-CONCENTRATED / DISTRIBUTED]
```

Maximum 6 lines.
No interpretation. No recommendation.
Distribution only.

---

## 6. ACTIVATION

Active in every report automatically.
If signal count is zero → all zones reported as LOW.
User is not separately notified.

---

## 7. HARD LOCK

This kit:
  Does not alter SECTION content
  Does not generate new signals
  Does not produce "weak/strong" judgments
  Only maps distribution
  Does not break UNIFIED-OUTPUT format

---

## 8. CANONICAL DEFINITION

ARGUMENT-DENSITY v1.0 aggregates signal load from all SECTION outputs
by functional zone and maps how claim weight is distributed
across the manuscript, without producing interpretation or judgment.

## 0. IDENTITY

This kit locks which overlay lens connects to which SECTION
and how each lens integrates within the CLASR-EN output.

---

## 1. DESIGN PRINCIPLE

Lens overlays do not operate independently.
Every lens is bound to a specific SECTION's sub-layer.

```
SECTION production
        ↓
Is a lens bound to this SECTION?
    Yes → Lens runs signal scan
           Signal found → integrated into SECTION output
    No  → Lens inactive for this SECTION
```

Lens output never opens a new section.
Always appears within the relevant SECTION as a sub-signal.

---

## 2. LENS — SECTION CONNECTION MAP

### Verbal Lens (VERBAL_LENS_KIT)
**Primary:** SECTION 6 — Academic Language & Hedging
  Certainty escalation → expression level flag in SECTION 6

  Impact inflation → "potential visibility" reframing in SECTION 6

## 3. COLLISION MANAGEMENT

When the same signal is detected by multiple lenses,
it is reported once, in the most relevant SECTION.

Examples:
```
Verbal Lens: "demonstrates" → certainty escalation (SECTION 6)
FREQUENCY SIGNAL: "demonstrates" → HIGH frequency (SECTION 6)
→ Same sentence / same SECTION → combined note, one appearance

LENSSA-DEPTH: AMPLIFICATION CLUSTER (SECTION 5)
FREQUENCY SIGNAL: AMPLIFICATION CLUSTER (SECTION 6)
→ Different SECTIONs → each reported separately, briefly

Temporality Lens: "inevitability" signal → SECTION 0
Orientation Lens: "inevitability" in scope framing → SECTION 1
→ Both may fire; reported in their respective SECTIONs separately
→ Not merged unless in the same SECTION

Integrity Signal: outcome instrument intersection → SECTION 0
Temporality Lens: urgency framing → SECTION 0
→ Both may fire in SECTION 0; reported as separate sub-signals
→ Not merged

Citation Behavior: confirmatory pattern → SECTION 2
Silence Lens: alternative explanations suppressed → SECTION 7
→ Different SECTIONs → each reported separately
→ Citation Kit takes precedence on bibliographic pattern;
  Silence Lens takes precedence on argumentative silence

Abstract–Body Coherence: claim escalation gap → SECTION 0
Calibration-Deep: over-claim detection → SECTION (any)
→ Calibration-Deep governs expression correction
→ Coherence Kit reports structural gap separately; not merged

Replication Signal: code availability absent → SECTION 3
Numerical Hat Kit: DEPENDENCY_BREAK (reproducibility claim
without code availability) → SECTION 7
→ Numerical Hat takes precedence in SECTION 7
→ Replication Kit reports dimension profile in SECTION 3
→ Both active; not merged

Contribution Framing: primacy claim → SECTION 1
Orientation Lens: "only option" framing → SECTION 1
→ Both may fire in SECTION 1; reported as separate sub-signals
→ Orientation Lens takes precedence on scope-closure behavior;
  Contribution Kit takes precedence on novelty claim behavior

Contribution Framing: gap assertion unsupported → SECTION 1
Citation Behavior: unsupported claim zone → SECTION 2 or SECTION 4
→ Citation Kit takes precedence on bibliographic absence;
  Contribution Kit reports gap claim framing behavior
→ Combined note if same SECTION; otherwise separate

Discussion Scope Drift: causal drift → SECTION 4
Verbal Lens: certainty escalation → SECTION 4 or SECTION 6
→ Verbal Lens takes precedence on language-level signal;
  Scope Drift Kit reports structural drift behavior
→ Combined note if same sentence; otherwise separate

Discussion Scope Drift: scale drift → SECTION 4
LENSSA-DEPTH: scale-shift detection → SECTION 5
→ LENSSA-DEPTH takes precedence in SECTION 5;
  Scope Drift Kit reports drift in SECTION 4
→ Both active; not merged

Discussion Scope Drift: implication extension → SECTION 4
Calibration-Deep: normative language detection (§1.4) → SECTION (any)
→ Calibration-Deep governs expression correction;
  Scope Drift Kit reports structural implication pattern
→ Not merged; collision not surfaced to user

Discussion Scope Drift: conclusion new claim → SECTION 4
Abstract–Body Coherence: claim deflation/escalation → SECTION 0
→ Coherence Kit takes precedence on abstract–body distance;
  Scope Drift Kit reports Results–Discussion–Conclusion chain
→ Both active; reported in their respective SECTIONs
```

Rule: More specifically bound lens takes precedence.
Collision is not surfaced to the user.
