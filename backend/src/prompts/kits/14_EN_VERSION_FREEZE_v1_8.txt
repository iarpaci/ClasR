## 6. VERSION FREEZE DOCUMENT v1.8

### CHANGE LOG — v1.7 → v1.8

Added:

31_EN_OUTPUT_MODE_KIT_v1_0
  User-triggered output format switching across three modes:
  AUTHOR MODE — signal label + plain-language explanation
    + "A reviewer will ask:" layer. Visual separator between layers.
  REVIEWER MODE — label + section location only.
    Maximum compression. Structured blocks retained.
  ADVISOR MODE — prioritised signal + one-sentence academic
    contextualisation. Global priority numbering across full report.
  Activation: Turkish and English commands accepted.
    yazar modu / author mode
    hakem modu / reviewer mode
    danışman modu / advisor mode
  Default: inactive — standard output unchanged.
  Tier 2. Related kit: READER_MODEL_KIT_v1_0.
  Does not generate signals. Does not modify detection behavior.
  Does not suppress labels. Does not reorder sections.
  UNIFIED-OUTPUT KIT v1.1 label enforcement active in all modes.

### RATIONALE

Gap 7 — Output audience mismatch:
  CLASR-EN produces a single output format regardless of
  who is reading the report. An author reading their own manuscript
  needs different presentation than a reviewer or advisor.
  Reader Model Kit maps the manuscript's implied reader.
  No kit mapped the report's actual reader.
  → Output Mode Kit v1.0.
