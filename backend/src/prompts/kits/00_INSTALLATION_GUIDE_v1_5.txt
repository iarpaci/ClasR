━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Start a new conversation in the project.
Paste a short paragraph from an English academic manuscript.

Expected behavior:
  - System runs AUTO-Q and displays estimate in SECTION 0
  - Report produces SECTION 0 through SECTION 8 in order
  - ARGUMENT DENSITY block appears before closing
  - Closing block appears with disclaimer
  - No translation residue. No policy language. No judgment.

For partial input test:
  - Paste abstract only
  - Expected: Type A header, suppressed SECTIONs with scope notes

For revision round test:
  - Add "R1 revision" before pasting manuscript
  - Expected: Revision Round header, expanded SECTION 8

For integrity signal test:
  - Use a manuscript with a competing interests declaration
    where authors have financial ties to the outcome instrument
  - Expected: INTEGRITY SIGNAL DETECTED note in SECTION 0
    and/or SECTION 3

For model fit threshold test:
  - Use a manuscript that states a CFI/TLI threshold
    and reports values below that threshold
  - Expected: MODEL FIT SIGNAL flag in SECTION 3
    with SELF-STATED THRESHOLD VIOLATION label

For citation behavior test:
  - Use a manuscript with heavy self-citation or no counter-literature
  - Expected: CITATION SIGNAL DETECTED in SECTION 2

For abstract–body coherence test:
  - Use a manuscript where abstract uses "demonstrates"
    but body uses "suggests" throughout
  - Expected: COHERENCE GAP DETECTED in SECTION 0

For replication signal test:
  - Use a manuscript with no data availability statement
  - Expected: REPLICATION SIGNAL PROFILE in SECTION 3
    with DATA AVAILABILITY ABSENT

For contribution framing test:
  - Use a manuscript with "first study to..." or "novel contribution"
  - Expected: CONTRIBUTION SIGNAL DETECTED in SECTION 1
    with intensity label (LOW / MEDIUM / HIGH)

For discussion scope drift test:
  - Use a manuscript where Discussion extends well beyond Results scope
  - Expected: SCOPE DRIFT SIGNAL DETECTED in SECTION 4
    with magnitude label (MINOR / MODERATE / MAJOR)

Full reading report (default):
  → Paste the manuscript text. System runs automatically.

Q-variant override:
  → Add "Q1", "Q2", or "Q3" before or after the text.

Summary mode:
  → Add "short report" or "summary" to your request.

Structural mode (headings only):
  → Paste a heading list (Table of Contents).

Q-comparison:
  → Request "compare Q1 and Q2" to receive two sequential reports.

Partial input:
  → Paste any incomplete portion. System detects and adjusts automatically.

Revision round:
  → Add "R1", "R2", "revision", or "resubmission" before the text.
